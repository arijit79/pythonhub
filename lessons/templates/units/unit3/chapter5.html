Windows Instructions:-

  • Go to Python Releases for Windows
  • Download any release >= 3.6.0
  • Once Download is finished. Double click on the installer and click on Yes in the prompt. The installer for Python should launch. Make sure that Install Pip and Add Python to PATH is checked and then click install. Once installed click Finish.
  • You cna launch Python by opening the start menu and clicking on IDLE Python 3.[WHATEVER.VERSION]

Linux Instructions

Python comes installed on most Linux distributions including Ubuntu, Arch Linux, RHEL, Fedora but in case you want to re-install or update it or install if its not installl, you can use your package manager. Here are commands for a few distros 
# Ubuntu, Debian etc
sudo apt install python3

# RHEL, Fedora, CentOS etc
sudo dnf install python3

# Arch Linux, Manjaro etc
sudo pacman -S python3

Mac OS Instructions

  • Go to Python Releases for MACOS
  • Download any release >= 3.6.0
  • Once Download is finished. Double click on the installer and the installer for Python should launch. Click on install. Once installed click Finish.
  • You cna launch Python by opening the finder and clicking on IDLE.app
