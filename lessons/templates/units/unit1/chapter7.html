{% extends "base_chapter.html" %} {% block chapter %} Software in a computer can be divided into a cople of categories based on different parameters. So let's have a look at softwares based on a couple of parameters:-
Free or Paid
  • Free:- These are the software that are freely available and do not cost anything for their whole lifetime
  • Paid:- These software cost money either at the time of download or after installation

Proprietary or Open-source:-
  • Open-Source:- These software have their source code freely available for viewing, modifying and redistribution
  • Proprietary:- These software do not have their source code freely available for viewing, modifying and redistribution
Syste Software or Application Software ot Utility Software
  • System Software:- These software are those that are used by the computer itself. They include Operating Systems and Kernals
  • Application Software:- These software are those that are used by the user and pass their commands to the system software. Examples would be word processors, spreadsheet softwares etc.
  • Utility Software:- These software give utility to the computer for example protection against viruses or disk defragmentation tools
