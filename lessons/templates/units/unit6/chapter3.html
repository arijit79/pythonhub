{% extends "base_chapter.html" %} {% block chapter %} A dictionary has also many functions with it. Let's have a look at these with a short discription
  • clear(): Clear all items from the dictionary
  • copy(): Return a copy of the dictionary
  • rfromkeys(): Create a dictionary from a given list/set/tuple and given values
  • get(): Return the value of a key. Return None if not found
  • items(): A set-like object providing a view on dictionary's items
  • keys(): Return keys of a dictionary in a list
  • pop(): Remove specified key and return the corresponding value
  • popitem(): Remove and return some (key, value) pair as a tuple
  • setdefault(): Insert key with a value of default if key is not in the dictionary
  • update(): Update dictionary from dict/list/set/tuple
  • values(): A list returning all values of dictionary
So that's a look at functions related to a dictionary. Now you should move on to create your own functions {% endblock %}
