{% extends "base_chapter.html" %} {% block chapter %} A list also has certain functions associated with it. Here are all the functions with a basic discription
  • append(): Add a item to a list
  • clear(): Removes all the elements in the list
  • copy(): Returns a copy of the list
  • count(): Returns how many times a element occur in a list
  • extend(): Expand list by using another list, tuple or a set
  • index(): Returns the first occurence of the given item
  • insert(): Insert object before index
  • pop(): Remove and return item at index (default last)
  • remove(): Removes first occurence of the given object
  • reverse(): Returns a reversed version of the list
  • sort(): Sort the list
Tuples and sets also have most of these methods except for a few or may contain more including these. You can check it out by using the dir() function {% endblock %}
