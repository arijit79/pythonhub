Addition
Subtraction

Increment and Decrement


Let;s see how you can increment and decrement a variable's value 
z = 5
z = z + 1 		# Increment line
print(z)

# Result
6
But that's a longer way. A much shorter way is by using += together 
z += 1
print(z)

# Result
6
So that's a much shorter way. You can do decrement, multiplication and division in the same way by using their symbols and an equal sign. It expands to the above in the Increment Line from the first example. Now you should move on to the next chapter {% endblock %}
