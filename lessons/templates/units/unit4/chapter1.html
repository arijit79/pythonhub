{% extends 'base_chapter.html' %} {% block chapter %} Python has many builtin data types. Here is a list of all the data types that are builtin to Python with their short discription
  1. int:- It holds an intger value without having any decimal places
  2. float:- Holds a floating point decimal value
  3. str:- Holds any string or alphanumeric data
  4. list:- Holds a list of many values into one memory location. Similar to a C or C++ array
  5. set:- Holds a list of many values without any duplicates
  6. tuple:-Holds an immutable list of values
  7. dict:- Called a dictionary, holds a key value pair of data
  8. True:- A True value, evaluated when a condition is true
  9. False:- A False value, evaluated when a condition is false
  10. None:- A None value, similar to null in other languages. Note it is not equal to a 0 since the 0 has a data type of int as well as value, while None is just a data type
These are the builtin data types in Python, you can create a simple data type for yourself just by creating a class and initializing an object for it.
To get the data type of a variable simpily use the type() function over it 
x = 3.14
print(type(x))

# Result
<class 'float'>
