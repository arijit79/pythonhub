{% extends "base_chapter.html" %} {% block chapter %} In Python, we have two types of loops while loops, which we discussed earlier and for loops. A for loop takes an iterable and gets a iterator from the iterable which it then runs a next() function to get the next value. We will discuss iterables and iterators in the upcoming chapters but now let's make a for loop. For loop uses the following syntax 
for ITERATOR in ITERABLE:
    statements...
Just consider ITERATOR as just a variable for now that we can loop over. Let's make a for loop to print numbers between 0 to 5 
for i in range(6):
    print(i)

# Result
0
1
2
3
4
5
So as you can see it printed numbers from 0 to 5. You might be thinking what is that range() function. It is a generator built into Python that returns numbers from a given range. We said it to print numbers between 0 and 5. Note that e gave an argument of 6 to the range() function since it does not include 6. Also we did'nt said a start so it by default started from 0. Now toy also might be thinking what that i is, it is the iterator that we are talking about. In the next line we just printed the current value of i. Now let's talk about two statements that are used in loop control, that are break and continue, they can be used in any loop, let's see what they do
  • The break statement breaks a loop without getting the loop complete. For example
    • for i in range(5):
    print(i)
    if i == 2:
        break

# Result
1
2
    So as you can see that it stopped the loop before completion
  • The continue statement
    • helps to skip the current iteration and move to the next 
    for i in range(5):
    if i == 2:
        continue
    print(i)

# Result
1
3
4
5
    So it skipped 2 as we said in the conditional that if i is equal to 2 then don't do anything but move to the next iteration of the loop
