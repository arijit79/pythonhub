{% extends "base_chapter.html" %} {% block chapter %} Logical operators allow you to join two or mrre conditions together. There are three Logical operators in Python
  1. and:- Returns True only when all the conditions are True and returns False if anyone of the condition is False
  2. or:- Returns True if any of the conditions is True, False if all the conditions are False
  3. not:- Returns True if the given condition is False and vice versa
Let's see a practical example of the above operators 
print(1 == 1 and 1 < 2)
print(1 > 2 or 1 < 2)
print(not 1 == 1)

# Results
True
True
False
So that's a look at Python's logical operators, if you have come from other languages you may be used to && for and, || for or and !! for not but in Python we use the keywords instead. This increases readablity of code and makes it easier to understand.
